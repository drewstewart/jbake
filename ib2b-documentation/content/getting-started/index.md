# The Basics

These iB2B documents are generated by the `nbn-jbake` static page generation tool. All the documents are written using [Markdown](https://help.github.com/articles/markdown-basics/) which is intended to be easy-to-read and easy-to-write.

## iB2B Project Structure

Projects have the following structure

```
.
|-- assets
|   |-- css
|   |-- fonts
|   |-- img
|   |-- js
|-- content
|   |-- about
|       |-- index.md
|   |-- services
|       |-- manage_product_order.md
|-- templates
|   |-- index.liquid
|   |-- interface-specification.liquid
|   |-- partials
|       |-- header.liquid
|-- jbake.properties
```
